
# Installation Guide: R and RStudio

## Step 1: Install R
R is the programming language.

👉 Download R from CRAN:  
[Download R (CRAN)](https://cran.r-project.org/)

Choose your operating system:
- Windows
- macOS
- Linux

Follow the installation steps (default settings are fine).

---

## Step 2: Install RStudio
RStudio is the interface that makes working with R easier.

👉 Download RStudio Desktop:  
[Download RStudio](https://posit.co/download/rstudio-desktop/)

Install it after installing R.

---

## Step 3: Open RStudio

After installation, open RStudio.

You will see:
- Script editor (top-left)
- Console (bottom-left)
- Environment (top-right)
- Files/Plots/Packages/Help (bottom-right)

---

## Step 4: Test your installation

Type this in the Console:

```r
2 + 2
