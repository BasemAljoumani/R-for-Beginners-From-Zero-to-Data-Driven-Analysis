# R for Beginners: From Zero to Data-Driven Analysis

*A solidarity-based course supporting Sudanese students and researchers.*

## About the Course

This repository contains the teaching materials, lesson scripts, exercises, and project resources for an introductory R course designed to support Sudanese students and early-career researchers.

The course is offered as a solidarity-based educational initiative to help participants build practical skills in R programming for data analysis, with a particular focus on environmental applications such as water and soil data.

## Course Structure

The course is structured into five progressive phases:

1. **Foundations**
2. **Data Structures and Tidyverse**
3. **Data Visualization**
4. **Working with Real Data**
5. **Statistical Analysis and Reproducible Workflows**

## Repository Structure

- `00_course_info/` – course announcement, syllabus, and schedule
- `01_phase_foundations/` – beginner lessons and exercises
- `02_phase_data_structures_tidyverse/` – data handling in R
- `03_phase_visualization/` – plotting and graphics
- `04_phase_real_data/` – real datasets and exploratory analysis
- `05_phase_statistics_reproducibility/` – regression, R Markdown, and Git
- `projects/` – mini-projects
- `slides/` – course slides
- `images/` – figures and screenshots

## Target Audience

Sudanese students and early-career researchers with little or no prior programming experience.
